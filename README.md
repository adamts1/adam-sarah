# הזמנה לחתונה – Wedding Invitation

Single-page wedding invitation with RTL Hebrew support.

## Tech

- **React** (Vite)
- **TailwindCSS**
- Responsive, RTL-ready

## Run

```bash
npm install
npm run dev
```

Open the URL shown in the terminal (e.g. `http://localhost:5173`).

## Build

```bash
npm run build
```

Output is in `dist/`.

## Hero image

Replace the placeholder in `src/components/Hero.jsx`:

- Set `HERO_IMAGE_PLACEHOLDER` to your Canva design image URL (e.g. after uploading to your host or CDN).

## Sections

1. **Hero** – Full-width background image, overlay, title, fade-in.
2. **אישור הגעה (RSVP)** – Form: name, number of guests, phone; validation and success message.
3. **פרשת שבוע** – Title, paragraph, decorative divider.

Design: white background, gold accents (`#C6A85E`), serif headings, rounded cards, soft shadows.
